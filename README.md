# Foreign Direct Investment

Interactive dashboard tracking Canada's net capital inflows and outflows — Foreign Direct Investment (FDI) flow data from Statistics Canada Table 36-10-0025-01.

## Features
- **Net capital flow** visualization (Inflow − Outflow)
- **Quarterly & Annual** granularity toggle
- **Country filter**: All Countries, United States, All Other Countries
- **Outflow component breakdown**: M&A, Reinvested Earnings, Other Flows
- **US vs Rest of World** destination comparison
- **Dark/Light theme** toggle
- Values in CAD Millions

## Data Source
Statistics Canada · Table 36-10-0025-01 · Balance of International Payments — FDI Flows (Quarterly)
